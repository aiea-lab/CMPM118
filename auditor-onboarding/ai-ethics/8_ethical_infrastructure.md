# Task 8: Ethical Infrastructure

The goal of this task is to create a reusable infrasture for testing ethics.
- [ ] Write a script in python that does the following:
  - [ ] Sends prompts to a model
  - [ ] Logs the output of the model (either locally or to a log file).
  - [ ] Tags failures using your rule.  (At this point, you can also add more rules to your rule set.)
The script should be self contained in that it takes in a prompt, and records if there is a failure or not without any human intervention.

## Deliverable
Write one page about your process of creating this script and upload it to Canvas.
