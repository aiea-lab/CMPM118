# Task 2: AI Ethics Onboarding

## Objective Your goal for this week is to:

1. Understand how AI systems are **documented and evaluated** through
model cards
2. Set up core tooling for LLM based experiments

## Part 1: Model Card Analysis

- [ ] Identify and read **at least 2 model cards** (e.g., [OpenAI](https://openai.com/index/gpt-oss-model-card/), HuggingFace, or other published models)

- [ ] For each model card, extract and document:
  - Intended use
  - Out-of-scope use
  - Evaluation metrics
  - Reported limitations
  - Ethical considerations (bias, safety, misuse)

- [ ] Compare the model cards:
  - What information is **well specified**?
  - What is **missing or underspecified**?
  - Are there gaps in evaluation (e.g., subgroup analysis, failure modes)?

## Part 2: Tooling Setup

- [ ] Set up access to the following tools:

### OpenAI API
- [ ] Request access to the lab OpenAI namespace
- [ ] Create an API key  
  - ⚠️ Do NOT commit your key to a public repository
- [ ] Go through at least **2 guides** from:
  - https://platform.openai.com/docs/overview
  - or other relevant tutorials
- [ ] Test that your key works by querying the model from a python file.

Write at least one page about the following and upload it to Canvas.

  - Summary of both model cards
