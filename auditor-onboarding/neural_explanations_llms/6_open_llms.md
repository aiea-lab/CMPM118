# Neural Explanations (LLMs) Project

- [ ]  This task can be completed either on Nautilus or on your own workstation
- [ ]  Download an Open Large Language Model of your choice
    - [ ]  For example, a list of some available open source models is reported here https://github.com/eugeneyan/open-llms
    - [ ]  I suggest to choose a model that can be run on your own machine for fast debugging
    - [ ]  Note: BERT based models (e.g., BERT, RoBERTa, distilBERT) are not considered large language models
- [ ]  Write a script to ask 9 questions to your LLM model
    - [ ]  3 questions should be open ended (long and open question, e.g., summary of an event)
    - [ ]  3 questions related to reasoning. You should provide the “rules” and “examples” of the task to your LLM for generating an answer and then ask a question relative to the task. This is called prompt engineering.
    - [ ]  3 questions related to reasoning but in this case the answer of the LLM needs to be a single word (e.g., you can use True/False questions, or Q/A)
- [ ]  Commit your changes to the github repo.
